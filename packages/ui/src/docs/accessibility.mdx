## Focus

- `--ds-focus-ring-color` and `--ds-focus-ring-width` are used for outlines.
- Use keyboard navigation to verify focus visibility in Storybook.

## ARIA

- Form fields use `aria-invalid` and `aria-describedby` when appropriate.
- Dialogs use `role="dialog"` and `aria-modal`.
